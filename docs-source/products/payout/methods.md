## 支持的 Payout 方式

Waffo Payout 支持多种资金发放渠道，商户可根据目标国家、时效要求和成本偏好灵活选择。

### 银行转账

#### 本地快速转账（Local Transfer）

通过目标国家本地清算网络发送资金，通常为 T+0 或 T+1 到账，手续费低于 SWIFT 国际电汇。

| 地区 | 清算网络 | 到账时效 |
|------|----------|----------|
| 中国 | CNAPS（大小额）| 实时 ~ T+1 |
| 美国 | ACH / Fedwire | T+1 ~ T+3 |
| 欧洲 | SEPA SCT | T+0 ~ T+1 |
| 英国 | Faster Payments | 实时 |
| 印度 | IMPS / NEFT | 实时 ~ T+1 |
| 新加坡 | FAST | 实时 |
| 澳大利亚 | NPP / OSKO | 实时 |
| 泰国 | PromptPay | 实时 |

#### SWIFT 国际电汇

适用于本地转账不覆盖的国家，通过 SWIFT 银行间网络跨境汇款。

- **到账时效**：T+2 ~ T+5 个工作日
- **覆盖范围**：200+ 国家和地区
- **适用场景**：大额跨境付款、供应商货款

### 数字钱包

| 钱包 | 覆盖地区 | 到账时效 | 单笔限额 |
|------|----------|----------|----------|
| Alipay（支付宝） | 中国大陆 | 实时 | ¥50,000 |
| WeChat Pay（微信支付） | 中国大陆 | 实时 | ¥50,000 |
| GrabPay | 东南亚 | 实时 | 视地区而定 |
| TrueMoney | 泰国 | 实时 | 50,000 THB |
| GCash | 菲律宾 | 实时 | 视账户级别 |

### 卡片退款（Card Refund）

将资金退回至原付款银行卡，适用于退款场景。

- **到账时效**：3 ~ 10 个工作日（取决于发卡行处理速度）
- **特点**：资金直接退至持卡人账户，用户体验好
- **限制**：金额不得超过原交易金额，部分卡组织限制发起时效

### 即时卡片到账（Push to Card）

通过 Visa Direct 或 Mastercard Send 将资金即时推送至持卡人的借记卡账户。

- **到账时效**：通常 30 分钟内
- **支持卡网络**：Visa、Mastercard
- **适用场景**：保险理赔、奖励发放、平台卖家收益

## 资金到账时效

| Payout 方式 | 最快到账 | 标准时效 | 说明 |
|-------------|----------|----------|------|
| 实时本地转账 | 即时 | < 1 分钟 | 支持国家限定 |
| 本地 ACH/SEPA | T+0 | T+1 | 工作日处理 |
| SWIFT 电汇 | T+2 | T+3~5 | 中间行可能额外收费 |
| 数字钱包 | 即时 | < 5 分钟 | 钱包账户须已激活 |
| 卡片退款 | T+3 | T+5~10 | 受发卡行处理速度影响 |
| Push to Card | < 30 分钟 | < 2 小时 | 需 Visa Direct 支持 |

<Info>
到账时效受收款行处理速度、节假日、汇率清算等因素影响，以上为参考值而非承诺时效。
</Info>

## 手续费说明

Waffo Payout 采用透明定价，商户可在控制台查看每笔 Payout 的预计手续费。

### 费用构成

| 费用类型 | 说明 |
|----------|------|
| Payout 服务费 | Waffo 收取的处理费用，按笔计费或按金额比例 |
| 中间行手续费 | SWIFT 跨境电汇中可能产生的中间行费用（第三方收取） |
| 汇率换算费 | 涉及货币兑换时，按实时汇率加价差计算 |
| 退款 | 对于失败的 Payout，服务费将在下一结算周期退还 |

### 费率区间参考

| Payout 方式 | 费率区间 |
|-------------|----------|
| 本地银行转账 | 0.3% ~ 0.8%（含最低手续费） |
| SWIFT 国际电汇 | 固定费 + 0.5% ~ 1.5% |
| 数字钱包 | 0.2% ~ 0.6% |
| 卡片退款 | 原交易手续费不退还 + 退款处理费 |

<Note>
实际费率取决于商户合同条款、Payout 量及目标国家。请联系 Waffo 商务团队获取专属报价。
</Note>
