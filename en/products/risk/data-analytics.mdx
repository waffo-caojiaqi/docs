## Subscription dashboard

The Waffo subscription dashboard provides merchants using subscription payments with a visual view of key business metrics:

- **MRR/ARR**: Trend charts for monthly/annual recurring revenue
- **Subscription volume**: Time-series data for active subscriptions, new subscriptions, and cancellations
- **Renewal rate**: Renewal success rate analysis by subscription plan
- **Churn analysis**: Breakdown of subscription cancellation reasons (voluntary cancellation, payment failure, expiration, etc.)
- **LTV analysis**: Customer lifetime value estimation

Merchants can filter data by time range, subscription plan, Channel, and more, and can export raw data for custom analysis.

## Error code analysis

Payment failures are a key factor affecting revenue. Waffo’s error code analysis helps merchants identify and resolve high-frequency failure issues:

- **Error code distribution**: Visualize the frequency and share of each error code
- **Trend analysis**: Time trends for specific error codes to quickly detect anomalous spikes
- **Channel comparison**: Side-by-side comparison of error rates across different payment methods and Channels
- **Error code descriptions**: Detailed explanations and recommended remediation for each error code

With error code analysis, merchants can quickly determine whether failures are caused by issuer declines, user input errors, or network issues, and optimize the payment flow accordingly.

## Checkout success rate analysis

Checkout success rate analysis provides end-to-end funnel data across the entire payment journey:

- **Funnel analysis**: Conversion rates for each step from entering checkout to completing payment
- **Payment method comparison**: Compare success rates and average processing time across payment methods
- **Regional analysis**: Distribution of payment success rates by the customer’s country/region
- **Time-of-day analysis**: Success-rate fluctuations by time period to identify performance issues during peak traffic
- **A/B testing**: Supports A/B testing for checkout UI configurations to optimize conversion with data-driven insights
