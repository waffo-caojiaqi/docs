## 服务介绍

Waffo Payout（代付）是面向平台商户的资金发放服务，支持商户将资金快速、安全地发送至全球收款方。无论是向用户退款、向供应商付款，还是向平台卖家分润，Waffo Payout 均提供统一的操作界面和 API 接口。

Payout 服务支持单笔发起和批量处理两种模式，覆盖银行转账、数字钱包、卡片退款等多种资金流向路径，并提供完整的到账状态追踪与对账报表。

## 适用场景

### 平台卖家分润

电商平台、内容平台等多边市场，定期向卖家、创作者结算收益。Waffo Payout 支持按结算周期批量发放，自动处理币种转换和手续费扣除。

### 供应商货款支付

跨境贸易场景下，向海外供应商支付货款。支持 SWIFT 电汇和本地快速转账两种路径，可根据时效和成本灵活选择。

### 用户退款与奖励发放

对于退款处理、用户奖励（返现、积分兑换）等场景，Payout 提供比原路退款更灵活的资金发放方式，支持发送至与支付时不同的账户。

### 员工薪资代发

跨国团队的薪资发放，支持多货币结算，自动将薪资按当日汇率换算并发至指定银行账户。

### 保险理赔

保险公司向理赔用户快速发放赔付金，支持银行卡即时到账和银行转账两种路径。

## 核心功能模块

| 功能 | 说明 |
|------|------|
| 单笔 Payout | 通过 Portal 或 API 发起单笔资金发放 |
| 批量 Payout | 上传 CSV 或通过 API 批量发起，支持异步处理 |
| Payout 查询 | 按状态、金额、收款方等维度查询历史 Payout 记录 |
| 收款方管理 | 创建和管理收款方（Recipient）信息，支持复用 |
| 汇率管理 | 实时汇率展示，支持锁汇操作 |
| 对账报表 | 按日/月导出 Payout 明细，与财务系统无缝对接 |
