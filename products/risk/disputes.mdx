## 拒付预警

拒付（Chargeback）是指持卡人通过发卡行对已完成的交易提出异议，要求撤销资金。一旦拒付成立，商户不仅损失交易金额，还需支付拒付处理费用，且高拒付率会影响商户资质和手续费率。

### 拒付预警服务（Chargeback Alert）

Waffo 接入 Visa 和 Mastercard 的官方拒付预警网络，在持卡人正式提起拒付前向商户发出提前预警。

**工作原理**：
1. 持卡人联系发卡行表达支付异议
2. 发卡行通过预警网络向 Waffo 推送预警信息
3. Waffo 在商户收到正式拒付通知前 **24~72 小时** 发出预警
4. 商户在预警期内完成退款，可撤销拒付申请

### 预警处理选项

| 处理方式 | 说明 | 适用场景 |
|----------|------|----------|
| 自动退款 | 收到预警后自动退款，100% 撤销拒付 | 商户不希望处理争议 |
| 手动审核 | 人工评估是否需要退款 | 商户需要判断合理性 |
| 忽略预警 | 不处理，等待正式拒付流程 | 商户确信交易合理 |

<Warning>
忽略拒付预警将导致正式拒付流程启动。建议对于金额较小或无证据的交易选择主动退款，避免拒付率升高影响商户账号健康。
</Warning>

### 拒付率监控

Waffo 实时计算并展示商户的拒付率指标：

| 指标 | 标准 | 说明 |
|------|------|------|
| Visa 拒付率 | < 0.9% | 超过 1% 进入 VDMP（监控计划） |
| Mastercard 拒付率 | < 1.5% | 超过 1.5% 进入 ETDP（高风险计划） |
| 拒付金额比率 | < 1% | 按金额口径计算 |

## 拒付处理

### 拒付流程概述

<Steps>
  <Step title="持卡人发起异议">
    持卡人通过发卡行渠道（APP、客服）对交易提出异议，说明拒付原因。
  </Step>
  <Step title="发卡行审核并发起拒付">
    发卡行审核持卡人异议，认为合理后通过卡组织网络向收单行（Waffo）发起正式拒付。
  </Step>
  <Step title="Waffo 通知商户">
    Waffo 在收到拒付请求后的 **3 个工作日内** 通知商户，并提供响应截止时间（通常为 7~20 天）。
  </Step>
  <Step title="商户提交证据">
    商户通过 Waffo 控制台或 API 上传争议证据（参见下方证据清单），在截止时间前提交。
  </Step>
  <Step title="卡组织裁决">
    发卡行和卡组织审查双方证据，作出最终裁决。裁决结果一般在 30~60 天内产生。
  </Step>
  <Step title="资金返还或确认损失">
    若商户胜诉，拒付金额退回商户账户；若商户败诉，拒付金额确认为损失。
  </Step>
</Steps>

### 有效争议证据

不同拒付原因码对应的有效证据不同：

| 拒付原因 | 有效证据 |
|----------|----------|
| 未收到商品/服务 | 物流跟踪记录、数字内容访问日志、服务交付证明 |
| 未授权交易 | 3DS 验证记录、设备指纹、IP 地址、用户行为日志 |
| 商品与描述不符 | 商品描述截图、与持卡人的沟通记录 |
| 取消订单未退款 | 退款处理记录、取消政策说明页面截图 |
| 重复扣款 | 交易明细，证明仅扣款一次 |
| 信用未处理 | 退款流水记录 |

## 争议处理流程

### 争议中心（Disputes Dashboard）

Waffo 控制台提供集中化的争议管理界面：

- **案件列表**：按状态（待响应、处理中、已裁决）筛选所有争议案件
- **截止时间提醒**：自动计算证据提交截止时间，支持邮件提醒
- **证据上传**：支持 PDF、图片、日志文件多格式上传，最大 50MB
- **历史记录**：完整保存所有争议沟通记录和证据文档

### 自动化争议响应

对于拥有大量交易的商户，Waffo 支持基于规则的自动争议响应：

```json
{
  "auto_respond_rules": [
    {
      "condition": "amount < 50",
      "action": "auto_refund",
      "reason": "低金额自动退款，避免拒付成本"
    },
    {
      "condition": "has_3ds_authentication == true",
      "action": "auto_submit_evidence",
      "evidence_types": ["3ds_authentication", "device_fingerprint"]
    }
  ]
}
```

### 争议结果分析

Waffo 提供争议结果统计报表，帮助商户优化争议策略：

| 分析维度 | 说明 |
|----------|------|
| 争议胜率 | 按支付方式、拒付原因码分维度统计赢率 |
| 拒付原因分布 | 识别主要拒付原因，针对性改进业务流程 |
| 拒付金额趋势 | 追踪月度拒付金额变化，及时发现异常 |
| 预警转化率 | 拒付预警响应率和成功撤销率 |

<Tip>
降低拒付率的最有效方法是主动预防：清晰的商品描述、及时的物流更新、便捷的退款政策，以及必要时的 3DS 强验证，都能显著减少持卡人异议的产生。
</Tip>
