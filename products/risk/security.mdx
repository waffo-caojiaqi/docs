## 数据与支付安全

Waffo 以安全为基础构建整个支付体系。从数据传输到存储，从交易验证到身份认证，每个环节均采用业界最高标准的安全技术，确保商户和用户的资金与数据安全。

### 传输安全

- **TLS 1.2 / 1.3**：所有 API 通信强制使用 TLS 加密传输，防止中间人攻击
- **证书固定（Certificate Pinning）**：移动 SDK 实现证书固定，防止证书伪造
- **HSTS**：HTTP Strict Transport Security，强制浏览器使用 HTTPS 连接

### 数据存储安全

- **AES-256 加密**：静态敏感数据（卡号、银行账户信息）采用 AES-256 算法加密存储
- **令牌化（Tokenization）**：原始卡号不落入商户系统，Waffo 生成不可逆的 Token 替代存储
- **密钥管理**：使用 HSM（硬件安全模块）管理加密密钥，定期轮换，最小权限访问
- **数据隔离**：商户数据严格隔离，多租户架构确保数据不互串

### 实时欺诈检测

Waffo 内置机器学习驱动的实时风控引擎，在交易完成前对每笔交易进行多维度风险评估：

| 检测维度 | 说明 |
|----------|------|
| 设备指纹 | 识别设备特征，检测设备异常或克隆 |
| 行为分析 | 分析用户操作行为模式，识别机器人 |
| 地理位置 | 检测异常地理跳跃（如短时间内不同国家登录） |
| 速度规则 | 识别短时间内高频交易（卡测试攻击） |
| 黑名单 | 对已知欺诈邮箱、IP、设备实时拦截 |
| 网络图谱 | 通过关联分析识别组织化欺诈团伙 |

## PCI、GDPR 等认证

### PCI DSS Level 1

Waffo 通过了 **PCI DSS Level 1** 认证——支付卡行业数据安全标准的最高级别，适用于每年处理超过 600 万笔交易的机构。

**认证意义**：
- 商户将支付数据传输至 Waffo 处理，无需自行维护 PCI DSS 合规
- 商户只需完成 **SAQ A**（最简级别）的自我评估，大幅降低合规成本
- 每年由 QSA（合格安全评估机构）进行独立审计

### GDPR 合规

Waffo 严格遵守欧盟通用数据保护条例（GDPR）：

- **数据处理协议（DPA）**：Waffo 作为数据处理者与商户签订 DPA
- **数据主体权利**：支持数据访问、更正、删除（被遗忘权）等请求处理
- **数据本地化**：欧盟用户数据存储在欧盟境内数据中心
- **隐私设计（Privacy by Design）**：数据最小化收集，默认最高隐私保护

### 其他认证

| 认证 | 说明 |
|------|------|
| SOC 2 Type II | 安全性、可用性、数据完整性第三方审计 |
| ISO 27001 | 信息安全管理体系国际标准 |
| 3DS 2.x 认证 | EMVCo 3-D Secure 协议认证 |
| PSD2 / SCA | 欧盟支付服务指令强客户认证合规 |

## 内部合规 / 风控机制展示

### AML / KYC 合规

Waffo 实施严格的反洗钱（AML）和客户身份验证（KYC）程序：

- **商户尽调（KYB）**：商户入网前完成业务背景调查，验证营业执照、受益所有人等信息
- **交易监控**：实时监控所有交易，对超过合规阈值的交易自动触发人工审核
- **可疑交易报告（SAR）**：发现涉嫌洗钱活动时向监管机构提交报告
- **制裁名单筛查**：对所有交易方实时比对 OFAC、EU、UN 等制裁名单

### 风控规则引擎

商户可通过 Waffo Dashboard 配置自定义风控规则：

```text
IF 单笔金额 > 10,000 USD
AND 用户账龄 < 30 天
THEN 触发 3DS 验证

IF 24小时内同一卡号交易次数 > 5
THEN 阻断交易并发送警报
```

### 安全运营中心（SOC）

Waffo 安全运营中心 7×24 小时运行，对平台安全事件进行实时监控和响应：

- 平均告警响应时间（MTTD）：< 5 分钟
- 安全事件处置时间（MTTR）：< 30 分钟
- 定期渗透测试：每季度由第三方安全机构执行

## 用户数据与隐私保护政策

### 数据收集原则

Waffo 遵循数据最小化原则，仅收集完成支付服务所必要的数据：

| 数据类型 | 收集目的 | 保留期限 |
|----------|----------|----------|
| 支付卡信息 | 交易处理（令牌化存储） | 不超过 5 年 |
| 交易记录 | 对账、争议处理、合规 | 7 年（监管要求） |
| 设备信息 | 欺诈检测 | 13 个月 |
| IP 地址 | 风险评估 | 6 个月 |

### 数据共享

Waffo 仅在以下情况下共享用户数据：
- 完成支付处理所必要的合作机构（银行、卡组织）
- 法律要求的监管机构信息披露
- 商户明确书面授权的第三方服务

**Waffo 承诺**：不将用户数据出售给第三方，不用于广告定向投放。
