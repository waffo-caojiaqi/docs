## 服务介绍

跨境电商和数字服务的税务合规是全球商户面临的重大挑战。随着各国对数字服务税（DST）和增值税（VAT/GST）监管日趋严格，商户需要针对不同国家的用户正确计算、收取和申报税款。

Waffo Tax 服务帮助商户自动化处理跨境支付场景下的税务合规工作，从税率计算到发票生成，从代扣代缴到申报辅助，全流程一体化处理。

## 收税

### 自动税率计算

Waffo 根据以下要素自动计算应收税款：

| 要素 | 说明 |
|------|------|
| 用户所在国家 | 根据账单地址、IP 地址或用户声明确定 |
| 商品/服务类型 | 数字服务、实物商品、混合供应各有不同税率 |
| B2C / B2B | 消费者交易与企业间交易税务处理不同 |
| 商户注册情况 | 商户是否在用户所在国完成 VAT 注册 |

### 支持的税种

| 税种 | 地区 | 说明 |
|------|------|------|
| VAT（增值税） | 欧盟、英国 | 数字服务适用 OSS / 原产地原则 |
| GST（商品和服务税） | 澳大利亚、新加坡、加拿大 | 境外数字服务提供商须注册 |
| 消费税 | 日本 | 10% 标准税率，数字服务适用 |
| 预扣税（WHT） | 多国 | 对非居民商户的服务费预扣 |
| 数字服务税（DST） | 法国、英国、西班牙 | 部分国家对数字服务额外征税 |

### 税款在收银台的展示

商户可配置税款展示方式：

```json
{
  "tax_display": "inclusive",  // 含税价，税款包含在展示金额内
  // 或
  "tax_display": "exclusive",  // 不含税价，结账时加收税款
  "tax_id_collection": true    // 是否收集 B2B 买家税号
}
```

### B2B 税号验证

针对企业客户，Waffo 支持在结账时收集买家 VAT 号，并通过 VIES 系统实时验证有效性。验证通过的 B2B 交易可适用反向征税机制（Reverse Charge），免除商户代扣义务。

## 报税

### 税务报表

Waffo 提供完整的税务报表，支持商户完成自主申报或交由税务顾问处理：

| 报表 | 更新频率 | 内容 |
|------|----------|------|
| 按国家税务汇总 | 月度 | 每个国家的应税交易金额和税款汇总 |
| 交易级税务明细 | 实时 | 每笔交易的税率、税款和适用规则 |
| 发票汇总 | 月度 | 所有已生成的税务发票 |
| VAT 申报辅助报表 | 季度/月度 | 按 EU OSS / 本地申报格式整理 |

### 电子发票生成

Waffo 可根据商户配置自动为每笔交易生成合规的电子税务发票：

- **发票语言**：支持 20+ 种语言
- **发票格式**：PDF（标准）、XML（电子发票格式）
- **发票内容**：商户信息、买家信息（含税号）、商品描述、税率、税款
- **自动发送**：支付完成后自动通过邮件发送给买家

### 支持的申报制度

| 申报制度 | 适用地区 | Waffo 支持 |
|----------|----------|------------|
| EU OSS（一站式申报） | 欧盟 | 提供申报汇总数据 |
| EU IOSS | 欧盟进口低值商品 | 申报辅助数据 |
| UK MTD VAT | 英国 | API 直连 HMRC |
| AU GST 季报 | 澳大利亚 | 申报明细报表 |
| JP 消费税申报 | 日本 | 申报明细报表 |

<Note>
Waffo Tax 提供税务数据支持和申报辅助，但最终申报责任由商户或其指定税务顾问承担。建议商户在开展新市场业务前咨询当地税务专业人士，确认合规要求。
</Note>
