## 功能概览

Waffo 商户 Portal 是商户管理日常业务的核心操作界面，提供交易查询、资金管理、报表下载、收银台配置等一站式功能。Portal 支持多用户、多角色权限管理，适合不同规模的运营团队协作使用。商户可通过浏览器随时访问，无需安装任何客户端软件。

## 账户管理

Portal 账户管理模块支持：

- **子账户管理**：创建多个子账户并分配不同权限（管理员、财务、运营、只读等角色）
- **API Key 管理**：生成和管理沙盒/生产环境的 API Key，支持一键轮换
- **Webhook 配置**：配置和测试 Webhook 端点，查看回调历史记录
- **KYB 信息**：查看和更新企业资质信息，跟踪 KYB 审核进度

## 交易查询

商户可在 Portal 实时查询所有支付交易：

- 按时间范围、交易状态、金额区间、支付方式等多维度筛选
- 查看单笔交易的完整详情（用户信息、支付方式、风控结果、手续费等）
- 对符合条件的交易发起退款操作
- 导出交易明细报表（CSV/Excel 格式）

## 结算管理

结算管理模块提供资金流向的完整视图：

- 查看当前待结算余额和历史结算记录
- 手动发起提现，指定金额和收款账户
- 下载结算单和对账文件
- 管理收款银行账户信息

## 设置

Portal 设置模块涵盖商户级别的全局配置：

- **收银台定制**：Logo、主题色、支持的支付方式、多语言配置
- **通知设置**：邮件告警阈值配置（如拒付率超标、余额不足）
- **风控规则**：配置自定义风控策略（需联系 Waffo 支持团队）
- **发票设置**：配置交易回执邮件模板和收件人
