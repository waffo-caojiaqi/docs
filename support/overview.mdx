## 常见问题

### 账户与入网

**Q: 注册后多久可以开始接受真实支付？**

A: 完成 KYB（商户入网审核）通常需要 2-5 个工作日。审核通过后，您的账户将自动升级为生产模式，可立即接受真实支付。

**Q: KYB 审核需要提交哪些材料？**

A: 通常需要：企业营业执照、法人身份证明、公司章程或股权结构说明、银行账户信息（用于结算）。具体要求因注册国家和业务类型而异。

**Q: 可以在沙盒环境下测试所有功能吗？**

A: 是的，沙盒环境支持所有 API 功能，包括支付、退款、订阅、Payout 等。唯一区别是不会产生真实资金流动。

### 支付与退款

**Q: 支持哪些货币？**

A: Waffo 支持 60+ 种货币，包括 USD、EUR、GBP、JPY、CNY、HKD、SGD、THB 等主流货币。具体支持列表请联系客户成功经理。

**Q: 退款需要多久到账？**

A: 退款处理时间因支付方式而异：信用卡退款通常需要 5-10 个工作日；数字钱包退款通常在 1-3 个工作日内完成。

**Q: 支付失败怎么排查？**

A: 首先查看 PaymentIntent 的错误详情（`error.code` 和 `error.message`），参阅[错误码查询](/developer/tools/error-codes)。如果是 `card_declined`，建议引导用户联系发卡行或使用其他支付方式。

### 技术集成

**Q: 如何保证支付不重复？**

A: 使用幂等键（Idempotency Key）：在创建 PaymentIntent 时传入唯一的幂等键，重复请求将返回已有对象而不会创建新的。

**Q: Webhook 事件未收到怎么处理？**

A: 检查：①Webhook 端点 URL 是否可公网访问；②是否正确验证了签名；③查看商户后台「Webhook 日志」中的发送记录和失败原因。

**Q: 支持哪些服务端语言？**

A: 官方提供 Node.js、Python、Go、Java、PHP、Ruby 的 SDK。其他语言可直接调用 REST API，参阅 [API 集成指南](/developer/integration/api-integration)。
